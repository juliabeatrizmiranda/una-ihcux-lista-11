O ReCiclo é um aplicativo que facilita a doação e o reaproveitamento de itens, conectando pessoas próximas e reduzindo o desperdício.
O protótipo possui seis telas: a Home- onde o usuário visualiza itens; Filtros- para refinar a busca; Detalhes do Item- com informações e contato; Cadastro- para adicionar novos itens; Chat- para comunicação entre usuários e Perfil- com dados, atividades e selos.
Na jornada principal, o usuário acessa a Home, busca ou filtra itens, vê os detalhes e entra em contato pelo chat. Para doar, cadastra o item e aguarda interessados.
